# ⏱ Chronos Pomodoro

Temporizador baseado na técnica Pomodoro, desenvolvido com **React**, **TypeScript** e **Vite**.

## 🎨 Tema Visual

- **Paleta:** Preto · Roxo · Branco
- **Tipografia:** Space Grotesk (UI) + JetBrains Mono (Timer)
- **Efeitos:** Gradientes, glow roxo, glassmorphism, hover animado

## 🚀 Como rodar

```bash
npm install
npm run dev
```
